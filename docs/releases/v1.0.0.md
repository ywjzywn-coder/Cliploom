# Cliploom 1.0.0

Cliploom 1.0.0 是首个正式稳定版本，面向 macOS 14 及更高版本，支持
Apple Silicon 与 Intel Mac。

## 下载安装

1. 下载 `Cliploom-1.0.0-macOS-universal-unnotarized.dmg`。
2. 打开 DMG，将 Cliploom 拖入 Applications。
3. 首次启动时，在“应用程序”中右键 Cliploom 并选择“打开”。
4. 按应用引导授予辅助功能与屏幕录制权限。

详细步骤见
[安装指南](https://github.com/ywjzywn-coder/Cliploom/blob/v1.0.0/docs/INSTALL.md)。

## 正式版功能

- 使用 `Option+V` 打开文本、链接、图片和文件剪贴板历史。
- 使用 `Option+A` 启动窗口识别或自由框选截图。
- 支持矩形、箭头、画笔、文字、马赛克、取色和 PNG 输出。
- 支持本地 OCR、二维码与条形码识别。
- 支持搜索、分类、收藏、删除、键盘操作和面板位置记忆。
- 剪贴板、截图与识别数据全部保存在本机，不上传内容。

## 1.0.0 改进

- 修复截图工具栏完成按钮边缘无法点击的问题。
- 截图和常驻帧缓存改为使用显示器当前模式的物理像素尺寸。
- 4K、Retina、缩放显示和竖屏显示器的选区预览与输出更加清晰。
- 选区尺寸标签现在显示最终图片的真实像素尺寸。

## 已知限制

- 为降低截图启动延迟，Cliploom 会维护 ScreenCaptureKit 帧缓存，因此
  macOS 可能持续显示屏幕共享提示。这是系统隐私指示，不代表画面被上传。
- 当前 DMG 使用 ad-hoc 签名且未经过 Apple 公证。首次启动可能需要右键
  “打开”，覆盖更新后个别 macOS 版本也可能再次确认屏幕录制权限。
