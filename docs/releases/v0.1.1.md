# Cliploom 0.1.1 Developer Preview

这一版首次提供可直接下载的 macOS Universal DMG，支持 Apple Silicon 和 Intel Mac。

## 安装

1. 下载 `Cliploom-0.1.1-macOS-universal-unnotarized.dmg`。
2. 打开 DMG，将 Cliploom 拖入 Applications。
3. 首次启动时，在“应用程序”中右键 Cliploom 并选择“打开”。
4. 按应用引导授予辅助功能与屏幕录制权限。

详细步骤见
[安装指南](https://github.com/ywjzywn-coder/Cliploom/blob/main/docs/INSTALL.md)。

## 本版变化

- 提供 Apple Silicon 与 Intel 通用安装包。
- 固定 Bundle ID、应用名称和 designated requirement，尽量改善覆盖更新后的权限稳定性。
- README 改为安装使用指南。
- 新增可重复执行的未公证 DMG 打包脚本。

## 安全说明

此 DMG 使用 ad-hoc 签名，未经过 Apple Developer ID 签名和 Apple 公证。
macOS 首次打开时会提示无法验证开发者，这是当前发布方式的预期行为。

安装包不会绕过系统安全设置，也不会上传剪贴板、截图、OCR 或扫码内容。
若需要无警告的正式分发版本，仍需 Developer ID Application 证书和 Apple 公证。
