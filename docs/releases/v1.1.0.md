# Cliploom 1.1.0

Cliploom 1.1.0 聚焦截图识别体验、二维码交互和后台资源占用优化。
本版本最低支持 macOS 15，支持 Apple Silicon 与 Intel Mac。

## 下载安装

1. 下载 `Cliploom-1.1.0-macOS-universal-unnotarized.dmg`。
2. 打开 DMG，将 Cliploom 拖入 Applications。
3. 首次启动时，在“应用程序”中右键 Cliploom 并选择“打开”。
4. 按应用引导授予辅助功能与屏幕录制权限。

详细步骤见
[安装指南](https://github.com/ywjzywn-coder/Cliploom/blob/v1.1.0/docs/INSTALL.md)。

## 主要更新

- OCR 结果窗口改为左侧图片预览、右侧可编辑文本，并支持 macOS 原生
  Live Text 选择和复制左图文字。
- QR/条形码结果改为直接显示在图片预览上：链接二维码中心显示动态箭头，
  非链接或不支持内容显示禁止标记。
- 多二维码图片会通过重叠分块和局部放大提升识别率，并避免重复标记。
- 截图保存按钮改为先隐藏截图遮罩，再异步打开系统保存面板，避免看起来卡死。
- 截图预热不再常驻 ScreenCaptureKit 屏幕流，降低长时间后台运行时的
  CPU、GPU 和内存压力。
- 取色器在框选开始后会隐藏，避免选区结束后色值浮层残留。

## 已知限制

- 当前 DMG 使用 ad-hoc 签名且未经过 Apple 公证。首次启动可能需要右键
  “打开”，覆盖更新后个别 macOS 版本也可能再次确认屏幕录制权限。
- 微信小程序码等非标准码制不一定能被系统 Vision 条码识别解析。
- 本版本的“跨设备同步”指同步操作逻辑，不会自动同步剪贴板数据。
