# Cliploom 0.1.2 Developer Preview

这一版集中改善截图交互、OCR/二维码结果窗口和取色体验，并继续提供支持
Apple Silicon 与 Intel Mac 的 Universal DMG。

## 安装

1. 下载 `Cliploom-0.1.2-macOS-universal-unnotarized.dmg`。
2. 打开 DMG，将 Cliploom 拖入 Applications。
3. 首次启动时，在“应用程序”中右键 Cliploom 并选择“打开”。
4. 按应用引导授予辅助功能与屏幕录制权限。

详细步骤见
[安装指南](https://github.com/ywjzywn-coder/Cliploom/blob/main/docs/INSTALL.md)。

## 本版变化

- 右键与 `Esc` 均可立即退出截图，子窗口获得焦点时也能生效。
- 自动识别的窗口保持原始亮度，并使用更明显的高对比度边框。
- 截图热键优先使用最近的内存帧，减少首次截图等待，并可保留快捷键触发时会消失的二维码弹窗。
- 从 OCR 或二维码结果页再次截图时，先捕获可见结果，再结束旧会话。
- 取色提示精简为颜色样本与 HEX 值，`Command+C` 可直接复制色值。
- OCR 结果窗口支持自适应尺寸、可拖动分栏和比例记忆。
- 修复二维码重复识别时的会话清理与结果显示问题。

## 已知限制

- 为降低截图启动延迟，Cliploom 会维护 ScreenCaptureKit 帧缓存。macOS
  因此可能持续显示屏幕共享提示；这是系统隐私指示，不代表画面被上传。
- 当前 DMG 使用 ad-hoc 签名且未经过 Apple 公证。首次启动可能需要右键
  “打开”，覆盖更新后个别 macOS 版本也可能再次确认屏幕录制权限。

## 隐私与校验

剪贴板、截图、OCR、二维码识别和取色均在本机完成，不上传内容。
发布附件同时提供 `SHA256SUMS.txt`，可用于校验下载文件是否完整。
