# Cliploom 0.1.0 Preview

Cliploom 的第一个源码预览版，提供原生 macOS 剪贴板历史与截图工作流。

## 本版内容

- `Option+V` 打开文本、链接、图片和文件历史。
- 搜索、分类、收藏、删除、键盘选择和直接粘贴。
- `Option+A` 进行窗口选择或自由框选截图。
- 原始分辨率 PNG、标注、马赛克、保存和剪贴板输出。
- 完全本地的 OCR 与二维码/条形码识别。
- OCR 图片/文字分栏、可拖动分隔线和比例记忆。
- 简体中文与英文界面。

## 系统要求

- macOS 14 或更高版本
- 源码构建需要 Xcode

## 如何体验

1. 下载 GitHub 自动生成的 Source code 压缩包。
2. 使用 Xcode 打开 `PasteBox.xcodeproj`。
3. 选择 `PasteBox` Scheme 和 `My Mac`。
4. 按 `Command+R` 运行。

成功标志：菜单栏出现 Cliploom 图标，按 `Option+V` 可以打开剪贴板面板。

## 关于安装包

此 Release 不附带 macOS 安装包。当前环境尚未配置 Developer ID Application
证书，无法生成适合公开分发、通过 Apple 公证的二进制文件。请勿把本地临时签名
构建当作正式发行包传播。
